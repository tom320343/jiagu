DexProtector

Android APK DEX加固工具 - 将DEX文件加密存储到SO原生库中

功能介绍

DexProtector 是一款 Android APK 加固工具，通过将 DEX 字节码文件使用 AES-256-CBC 加密后嵌入到原生共享库 (.so) 中，在运行时从 native 层解密并通过 InMemoryDexClassLoader 动态加载，有效防止静态逆向分析和 DEX 反编译。

技术架构

DexProtector/
├── app/          # 加固工具 APP (Compose UI)
│   └── engine/   # 核心引擎
│       ├── ApkParser.kt      - APK解析
│       ├── DexEncryptor.kt   - AES-256-CBC加密
│       ├── SoPatcher.kt      - SO尾部注入
│       ├── ManifestPatcher.kt - Manifest修改
│       ├── ApkRepackager.kt  - 重新打包
│       └── ApkSigner.kt      - PKCS#7签名
└── stub/         # 注入目标APK的桩模块
    ├── cpp/      # C++ AES解密 + DEX加载
    └── java/     # StubApplication (ClassLoader替换)

构建命令

# 设置环境变量
export ANDROID_HOME=/opt/android-sdk
export ANDROID_NDK_HOME=/opt/android-sdk/ndk/26.1.10909125

# 构建 stub 原生库
./gradlew :stub:assembleRelease

# 拷贝 .so 到 assets
cp stub/build/intermediates/stripped_native_libs/release/out/lib/*/libdexprotector.so \
   app/src/main/assets/stub_libs/

# 构建 APK
./gradlew :app:assembleDebug
